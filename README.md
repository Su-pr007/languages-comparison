# Сравнение языков

Заводил проект, чтобы сравнить, как обработают 1 млн запросов разные языки программирования. Разницы оказалось никакой.

## Запуск проекта
```bash
docker compose up -d
```

Чтобы скомпилировать go вручную:
```bash
docker compose build go
```

## Запуск действий
Отправка 100 000 запросов через go:
```bash
time docker compose run --rm go 100000
```

Через PHP:
```bash
time docker compose run --rm php 100000
```
